# 쇼핑 리스트 앱

간단하고 직관적인 쇼핑 리스트 웹 애플리케이션입니다.

## 기능

- 쇼핑 아이템 추가/삭제
- 아이템 체크(완료) 표시
- 전체 삭제 기능
- 로컬 스토리지를 이용한 데이터 저장 (브라우저를 닫아도 유지)
- 반응형 디자인

## 사용 방법

1. `shopping-list.html` 파일을 브라우저에서 열기
2. 입력창에 아이템 이름 입력 후 "추가" 버튼 클릭 또는 Enter 키
3. 아이템 왼쪽 원형 체크박스를 클릭하여 완료 표시
4. 아이템 오른쪽 × 버튼으로 개별 삭제
5. "전체 삭제" 버튼으로 모든 아이템 삭제

## 기술 스택

- HTML5
- CSS3 (Flexbox, Gradient)
- Vanilla JavaScript
- LocalStorage API

## 스크린샷

보라색 그라데이션 배경에 깔끔한 흰색 카드 UI로 구성되어 있습니다.
